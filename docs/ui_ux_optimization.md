## 🔍 可优化的部分

### 1. **视频卡片样式** 🎬

**当前状态**：
- 使用边框 `border: 1px solid`
- hover 效果较简单

**建议优化**：
```css
.video-card {
    border: none;
    border-radius: 12px;
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.08);
    transition: all 0.3s ease;
}

.video-card:hover {
    transform: translateY(-4px);
    box-shadow: 0 8px 24px rgba(0, 0, 0, 0.12);
}
```

**理由**：与用户统计卡片保持一致的阴影层次风格

---

### 2. **下载历史面板** 📜

**当前状态**：
- 可能有边框样式
- 卡片之间层次感不足

**建议优化**：
- 历史项卡片使用阴影替代边框
- hover 时轻微上浮 + 阴影加深
- 添加缩略图圆角和阴影

---

### 3. **设置面板（Settings Drawer）** ⚙️

**当前状态**：
- 可能有边框分隔
- 输入框样式可能不一致

**建议优化**：
- 使用分隔线或留白替代边框
- 输入框添加柔和的阴影
- 主题切换按钮优化为圆角卡片式

---

### 4. **Toast 提示消息** 💬

**当前状态**：
- 可能有边框
- 层次感不足

**建议优化**：
```css
.toast {
    border: none;
    box-shadow: 0 4px 20px rgba(0, 0, 0, 0.15);

}
```

---

### 5. **Modal 弹窗** 🎭

**当前状态**：

- 添加背景模糊效果

### 6. **快捷卡片（Shortcut Cards）** 🚀

**当前状态**：
- 可能有边框样式
- 与整体风格不一致

**建议优化**：
- 改用阴影层次
- hover 时上浮 + 阴影加深

---

### 7. **下拉菜单（Dropdown）** 📋

- 边框明显
- 层次感不足

**建议优化**：

### 8. **输入框（Input Fields）** ✏️

- 默认状态：柔和的底部阴影

- 移除边框，使用背景色变化

---

### 9. **按钮样式** 🔘

**当前状态**：
- 可能有边框
- 层次感可以加强

**建议优化**：
- 主要按钮：使用渐变 + 阴影
- 次要按钮：边框改为柔和阴影
- hover：阴影加深 + 轻微上浮

---

### 10. **状态指示器** 🟢

**当前状态**：
- 简单的圆点
- 可能缺乏层次

**建议优化**：
- 添加柔和的光晕效果
- 动态呼吸动画

## 🎯 统一设计原则

### 阴影层次系统
```
Level 1: 0 2px 8px rgba(0, 0, 0, 0.08)     // 卡片默认
Level 2: 0 4px 16px rgba(0, 0, 0, 0.12)    // hover
Level 3: 0 8px 32px rgba(0, 0, 0, 0.15)    // 弹窗、Modal
Level 4: 0 12px 48px rgba(0, 0, 0, 0.2)    // 最高层级
```

### 间距系统
```
xs: 4px
sm: 8px
md: 12px
lg: 16px
xl: 24px
2xl: 32px
```

### 圆角系统
```
sm: 8px
md: 12px
lg: 16px
xl: 20px
full: 50%
```

## 💡 额外建议

### 动画与过渡
- 统一使用 `transition: all 0.3s ease`
- hover 上浮：`transform: translateY(-4px)`
- 点击下沉：`transform: scale(0.98)`

### 亮色/暗色模式
- 确保阴影在不同模式下都合适
- 亮色模式：使用更柔和的阴影
- 暗色模式：阴影可以更深

### 响应式优化
- 移动端：减小阴影范围
- 平板：中等阴影
- 桌面：完整阴影效果
